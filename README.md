Car Market Analysis Dashboard:
A data analysis project focused on exploring used car market trends using Python and Power BI.
The dataset was cleaned and transformed using Pandas, then visualized through an interactive Power BI dashboard featuring KPIs, pricing analysis, fuel type distribution, scatter plot analysis, and business storytelling insights.

Tools Used
* Python (Pandas)
* Power BI
* DAX

Key Insights
* Diesel cars showed the highest average prices.
* Luxury brands maintained premium pricing.
* Vehicle prices decreased as mileage increased.
* Economy brands dominated the market share.
